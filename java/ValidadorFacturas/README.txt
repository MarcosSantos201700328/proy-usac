La aplicación está hecha bajo jdk 1.5, pero preferentemente ejecutarlo bajo la  JRE versión 1.6.

Es necesario modificar los parámetros de conexión a la dbms que se vaya a utilizar, esto se hace en el archivo Query.java, en las variables miembro estáticas encontradas al inicio de la clase. Estos parametros son usuario, password, URI de conexion JDBC y el paquete de conexion para jdbc.
