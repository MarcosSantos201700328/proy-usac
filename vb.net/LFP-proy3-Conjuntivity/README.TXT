Conjuntivity

A set operation program

Requirements:
.NET framework 

Description:
This is a set operation program made in the first year of my career, it was made for the Formal Languages and Automata Course of the university. It use a descendent-recursive hardcoded parser to parse the input, and allows union, intersection, Difference of strings "elements" between the defined sets.

It was developed using Visual Studio .NET 2003

E.g

A= { aa,bb }
B = { bb, cc}
d = A U B
... and so on,
